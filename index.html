Toolbox

A collection of single-purpose utilities for the small web.

NEW: Filenames have been scrubbed and unified.
  • Bulk RSS Finder
    Sniff RSS/Atom feeds from a list of URLs.
  • URL List Cleaner
    Deduplicate, trim, and scrub large link lists.
  • Bulk URL Opener
    Open a list of links in multiple tabs at once.
  • HTML Code Editor
    Live preview and scratchpad for web snippets.
  • CSS Scrubber
    Strip styles and CSS from HTML to see the structure.
  • Paste to Markdown
    Convert formatted web content into clean Markdown.
  • Text Cleaner
    Remove empty lines and whitespace from text blocks.
  • ISBN Fetcher
    Retrieve ISBN-10/13 codes from book titles in bulk.
  • Book List Cleaner
    Scrub messy book lists into one-per-line titles.
  • CSV to JSON
    Convert CSV data into JSON or standard HTML tables.
  • Hex Visualizer
    Preview and verify HTML hex colors instantly.
  • Simple RSS Reader
    A lightweight browser-based feed viewer.
  • RSS to Markdown
    Turn feed entries into clean markdown notes.
  • ToC Generator
    Online markdown table of contents generator.
  • OMDb Lookup
    Search movie metadata and posters via API.
  • Guestbook
    A shared canvas for digital garden visitors.
  • Extra Tools
    Experimental scripts and niche utilities.
  • Changelog
    The history of pushes and tool updates.
