Toolbox

A collection of single-purpose utilities for the small web.

LATEST: Moved experimental markdown parsers to the /demo/ directory.

Book & Reading Tools

  • Batch ISBN Finder
    Fetch ISBN-10/13 codes from book titles.
  • Book List Cleaner
    Scrub messy lists into clean titles.
  • OPML Converter
    Inspect and export RSS subscription files.

RSS & Feed Utilities

  • Bulk RSS Finder
    Sniff feed links from website lists.
  • RSS Reader
    Browser-based viewer for RSS/Atom feeds.
  • RSS to Markdown
    Convert feed entries into markdown notes.
  • Widget Generator
    Create embeddable RSS widgets.

Markdown Demos (/demo/)

  • Claude Markdown-it
    Markdown-it parser optimized for Claude.
  • Jinja Markdown-it
    Markdown-it parser with Jinja support.
  • Markdown-it
    Standard markdown-it live editor.
  • URL to Markdown
    Convert web content directly to markdown.

Development & Code

  • HTML Editor
    Live scratchpad for web snippets.
  • Hex Visualizer
    Preview HTML hex color codes.
  • CSV to JSON
    Convert CSV data to JSON tables.
  • ToC Generator
    Generate Markdown Tables of Contents.
