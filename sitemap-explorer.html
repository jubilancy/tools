Sitemap Explorer

Paste sitemap XML to visualize URL clusters, depth levels, content families, and orphan patterns. Uses fast-xml-parser logic and Chart.js.

Note: browser fetch requires the sitemap server to allow CORS. If it fails, copy-paste the XML instead.

Tip: XML is parsed with the browser's built-in DOMParser. Sitemaps with <sitemapindex> (index files) list child sitemaps rather than URLs — paste individual sitemaps for full detail.
