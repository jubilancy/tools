URL to Markdown

Fetch any web page and convert it to clean Markdown.

Backends: Jina.ai (r.jina.ai) fetches most pages cleanly with no options needed. urltomarkdown uses Turndown + Mozilla Readability for more control, but runs on Heroku free tier and may be asleep. If a page blocks both, try the browser extension: Firefox · Safari.
