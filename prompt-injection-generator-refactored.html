How it works: This tool embeds hidden instructions within normal-looking text using Unicode control characters that are invisible to the human eye but can be parsed by LLMs. Different techniques use various Unicode categories to hide content. This is useful for understanding how LLMs can be manipulated through careful prompt construction.
