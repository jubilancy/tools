Extra

Tips, tricks, and additional resources for using the tools.

🎨 Theme Customization
Every tool has a theme toggle button in the top-right corner. Your choice is saved automatically. The theme preference persists across all tools and page reloads using your browser's localStorage.
📋 Copy-to-Clipboard
Most tools with text output have a "Copy" button. Click it to copy the result to your clipboard. Your browser will confirm with a status message. You can then paste it anywhere with Ctrl+V or Cmd+V.
🔒 Privacy & Security
All tools process data entirely in your browser. No data is sent to external servers (except when fetching RSS feeds from URLs you specify). Your privacy is protected. All processing happens locally on your device.
📱 Mobile Support
All tools are fully responsive and work on mobile devices. Buttons stack vertically on small screens. Text areas and inputs scale appropriately. Use landscape orientation for larger screens when available.
🌐 Browser Compatibility
These tools work in modern browsers: Chrome 90+ • Firefox 88+ • Safari 14+ • Edge 90+
For best experience, keep your browser updated. Old browsers may have limited support.
⌨️ Keyboard Shortcuts
Most tools accept Enter key submissions in input fields. Press Ctrl+A to select all text in a textarea. Use Tab to navigate between fields.
🔄 Bulk Processing
Tools like "Bulk Hyperlink Generator", "Bulk URL Generator", and "Bulk RSS Finder" process multiple items at once. Paste one item per line, then click the action button. Results appear instantly.
💾 Saving Your Work
Use your browser's "Save as" feature to download tool output. Most tools also have export buttons (CSV, JSON, etc.). Some tools like the Guestbook save data to localStorage automatically.
💡 Pro Tip: Bookmark your favorite tools for quick access. You can also create browser shortcuts on mobile.
🐛 Troubleshooting
Tool not loading? Try refreshing the page or clearing your browser cache.
Copy button not working? Use Ctrl+A to select all, then Ctrl+C to copy manually.
Theme not saving? Check if your browser has localStorage enabled. Try a different browser.
RSS feed not loading? The feed URL must be publicly accessible and valid RSS/Atom format.
📚 Feature Categories
Text Processing: Clean text, remove formatting, work with code
URL Tools: Clean, split, generate, and manage URLs
Data Conversion: Convert between formats (CSV, JSON, HTML, Markdown)
RSS/Feeds: Read, find, convert, and widget-ify RSS feeds
Collections: Manage lists, books, ISBNs, and guestbook entries
🔗 External Services
Some tools use external services for specific features:
AllOrigins: CORS proxy for fetching RSS feeds (allorigins.win)
Turndown/GFM: HTML to Markdown conversion libraries (unpkg.com)
These services are free and don't store your data.
❓ FAQs
Q: Are my data safe?
A: Yes. All processing happens in your browser. No servers store your data.

Q: Can I use these tools offline?
A: Most can! Download the HTML file and open locally. RSS tools need internet for feed fetching.

Q: Why is my theme not saving?
A: Check browser settings. Some browsers require localStorage permission. Try another browser if issues persist.

Q: Can I share these tools?
A: Yes! Share the link to tools.eliana.lol. The tools are free and designed for sharing.
