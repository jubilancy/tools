Data Visualizer

Turn CSV, JSON, or raw x/y data into charts, pivots, scatter plots, and bubble maps — one tool instead of nine.

Input Format
Chart Type
Title
Aggregation (CSV pivot mode)
CSV Data

Two-column CSV (Category,Value) plots directly. Three-column CSV (Column,Category,Value) uses the aggregation above to pivot.

JSON Data

Array of objects. Uses label/name and value keys if present, otherwise the first two keys of each object.

X,Y Pairs (one per line)
X-Axis Label
Y-Axis Label
Point Size
Bubble Data (Name, X, Y, Size, Color)

Color is optional hex, defaults to the accent color.

Map Width
Map Height
Tip: CSV mode with a third column (Column,Category,Value) automatically pivots your data — pick Sum, Average, Count, Max, or Min above. Bubble Map mode ignores the Chart Type selector and always renders an SVG bubble layout.
